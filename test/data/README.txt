This directory contains two files used in the unit tests:
 - static_output.txt: this is an output file generated by sambvca21 as provided by @andre-cloud which is known to cause
 issues when retrieving the percent free volume (see https://github.com/JacksonBurns/py2sambvca/issues/31)
 - nhc.xyz: taken from the sambvca examples website, this set of coordinates is used in the tests that dynamically run
 py2sambvca
